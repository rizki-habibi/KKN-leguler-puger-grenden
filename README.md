## 🚀 Cara Menjalankan (Lokal)

### Prasyarat
- Node.js v18+
- pnpm v9+
- MySQL / Laragon
- Git

### 1. Clone Repository
```bash
git clone https://github.com/rizki-habibi/KKN-leguler-puger-grenden.git
cd KKN-leguler-puger-grenden
```

### 2. Setup Environment
Buat file `.env` di root project:
```env
DATABASE_URL=mysql://root@127.0.0.1:3306/kkn_grenden
SESSION_SECRET=grenden-kkn-secret-2026
ADMIN_PASSWORD=grenden2025
PORT=5000
NODE_ENV=development
FRONTEND_PORT=3000
VITE_API_URL=http://localhost:5000
```

### 3. Setup Database
```bash
# Import tabel dan data awal
mysql -u root -h 127.0.0.1 -P 3306 < scripts/setup-db.sql
```

### 4. Install Dependencies
```bash
pnpm install --ignore-scripts
```

### 5. Jalankan Server

**Terminal 1 — Backend API:**
```bash
cd artifacts/api-server
DATABASE_URL=mysql://root@127.0.0.1:3306/kkn_grenden SESSION_SECRET=grenden-kkn-secret-2026 npx tsx src/index.ts
```

**Terminal 2 — Frontend:**
```bash
cd artifacts/grenden-desa
npx vite --port 3000
```

### 6. Buka Browser
- **Website:** http://localhost:3000
- **API:** http://localhost:5000/api/anggota
